# Schedule comparison: Sling vs Demo solver

Static comparison page for one week (2026-05-25 — 2026-05-31) of the Coffee
Bigshot schedule. Top grid shows the real Sling-scheduled shifts; bottom grid
shows what the solver produced when told to ignore Sling entirely.

Generated by `python3 build.py` from local snapshots `profiles.json`,
`demo.json`, `sling.json`. Output: self-contained `index.html`.

To refresh: rerun `build.py` after pulling fresh data from
`ivan.kuzelev.com/api/{profiles,draft/{week},sling/external?week={week}}`.

Page: https://ionfet.github.io/sling-vs-demo/
